# ✨ منزلق المنتجات - التوثيق الكامل

## 📌 ملخص المتطلبات المُنفذة:

✅ **10 منتجات فقط** - يتم جلب أول 10 منتجات من `products.json`
✅ **عرض 4 في المرة** - يتم عرض 4 منتجات بجانب بعضها
✅ **حركة سلايد** - حركة سلسة عند التنقل
✅ **واحد يمين وآخر يسار** - أزرار للتنقل الأمامي والخلفي
✅ **الضغط على اسم المنتج** - حدث تفاعلي عند الضغط
✅ **بدون ملف HTML منفصل** - تم التكامل مباشرة في الصفحة الرئيسية

---

## 🎯 كيفية الاستخدام:

### 1️⃣ من الصفحة الرئيسية (index.html):
افتح `index.html` وستجد منزلق المنتجات في الأعلى قبل قسم المنتجات المميزة.

### 2️⃣ للاختبار المنفصل:
افتح `test-slider.html` لاختبار المنزلق بشكل منفصل.

---

## 🔧 التفاعلات:

### 🖱️ الضغط على أزرار التنقل:
```
زر يسار (◄) = التنقل للشرائح السابقة
زر يمين (►) = التنقل للشرائح التالية
```

### 🖱️ الضغط على مؤشرات الشرائح:
- اضغط على أي مؤشر (النقاط في الأسفل) للانتقال مباشرة لتلك الشريحة

### 🖱️ الضغط على اسم المنتج:
- عند الضغط على اسم المنتج، يتم تفعيل حدث `onProductNameClick`
- يمكن إضافة منطق في `JS/product-slider.js` لتنفيذ أي شيء (مثل الانتقال لصفحة التفاصيل)

### ⏱️ الحركة التلقائية:
- تبدأ كل 6 ثوان تلقائياً
- تتوقف عند التفاعل اليدوي
- تستأنف بعد 6 ثوان

## 🚀 خيارات التخصيص:

في **JS/product-slider.js** بنهاية الملف:

```javascript
new ProductSlider({
    sliderId: 'productSlider',           // معرّف عنصر المنزلق
    prevBtnId: 'prevBtn',                // معرّف الزر السابق
    nextBtnId: 'nextBtn',                // معرّف الزر التالي
    indicatorsId: 'indicators',          // معرّف المؤشرات
    productsUrl: 'products.json',        // مسار ملف JSON
    itemsPerView: 4,                     // عدد المنتجات المعروضة
    autoPlay: true,                      // تفعيل التشغيل التلقائي
    autoPlayInterval: 6000               // الفاصل الزمني (ميلي ثانية)
});
```
