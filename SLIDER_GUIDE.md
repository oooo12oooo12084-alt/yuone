### 🎯 منزلق المنتجات - شرح سريع

```
┌─────────────────────────────────────────────────────────┐
│        أفضل المنتجات                                      │
├─────────────────────────────────────────────────────────┤
│  < │ [المنتج 1] [المنتج 2] [المنتج 3] [المنتج 4] │ >   │
│    │                                        │           │
│    │  عند الضغط على اسم المنتج يتحرك السلايد│           │
│    │  واحد يمين (السابق) وآخر يسار (التالي)│           │
│    └────────────────────────────────────────┘           │
│                                                          │
│     ● ◎ ◎ ◎  (مؤشرات الشرائح)                         │
└─────────────────────────────────────────────────────────┘
```

### 📋 حركات المنزلق:

**الضغط على اسم المنتج:**
- يتم تفعيل حدث `onProductNameClick`
- يمكن إضافة أي منطق كالانتقال لصفحة التفاصيل

**الضغط على الأزرار:**
- الزر الأيسر (◄): يعرض المنتجات السابقة
- الزر الأيمن (►): يعرض المنتجات التالية

**الضغط على المؤشرات:**
- الانتقال المباشر لأي شريحة

### 🔄 الحركة التلقائية:

- تبدأ تلقائياً كل 6 ثوان
- تتوقف عند التفاعل اليدوي
- تستأنف بعد 6 ثوان من آخر تفاعل
