## 🔧 التخصيص:

### تغيير سرعة الحركة التلقائية:

افتح: `JS/product-slider-new.js`

ابحث عن:
```javascript
autoPlayInterval: 10000  // 10 ثوانٍ
```

غيّر الرقم:
- `5000` = 5 ثوانٍ (أسرع)
- `10000` = 10 ثوانٍ (حالي)
- `15000` = 15 ثانية (أبطأ)
- `20000` = 20 ثانية (بطيء جداً)

---

### تغيير عدد المنتجات المرئية:

افتح: `JS/product-slider-new.js`

ابحث عن:
```javascript
itemsPerView: 4  // 4 منتجات مرئية
```

غيّر الرقم:
- `3` = 3 منتجات (أقل)
- `4` = 4 منتجات (حالي)
- `5` = 5 منتجات (أكثر)

---

### إضافة فئة جديدة (مثلاً: KEYBOARD):

افتح: `JS/product-slider-new.js`

أضف بعد قسم RAM:
```javascript
// KEYBOARD
new ProductSlider({
    sliderId: 'keyboardSlider',
    prevBtnId: 'keyboardPrevBtn',
    nextBtnId: 'keyboardNextBtn',
    indicatorsId: 'keyboardIndicators',
    productsUrl: 'products.json',
    categoryFilter: 'keyboard',
    productsCount: 10,
    itemsPerView: 4,
    itemsToScroll: 1,
    autoPlay: true,
    autoPlayInterval: 10000
});
```

ثم في `index.html` أضف قسم جديد:
```html
<!-- KEYBOARD Section -->
<section class="product-slider-section" style="margin-top: 40px; margin-bottom: 40px;">
    <div class="container">
        <h2 style="text-align: center; margin-bottom: 30px; font-size: 28px; color: #333;">⌨️ KEYBOARD - لوحات المفاتيح</h2>
        
        <div class="slider-container-new">
            <div class="slider-wrapper-new">
                <div class="slider-new" id="keyboardSlider">
                    <!-- المنتجات سيتم إدراجها هنا -->
                </div>
            </div>
            
            <button class="slider-btn-new prev-btn-new" id="keyboardPrevBtn">
                <i class="fas fa-chevron-right"></i>
            </button>
            <button class="slider-btn-new next-btn-new" id="keyboardNextBtn">
                <i class="fas fa-chevron-left"></i>
            </button>
            
            <div class="slider-indicators-new" id="keyboardIndicators"></div>
        </div>
    </div>
</section>
```

## 🐛 استكشاف الأخطاء:

### المشكلة: لا تظهر المنتجات

**الحل:**
```
1. تأكد من وجود products.json في المجلد الرئيسي
2. افتح Console (F12) وابحث عن أخطاء
3. أعد تحميل الصفحة (Ctrl+F5)
```

### المشكلة: الحركة التلقائية لا تعمل

**الحل:**
```
1. تحقق من autoPlayInterval (يجب أن يكون > 0)
2. تأكد من أن autoPlay = true
3. جرب متصفح مختلف
```

### المشكلة: الأسهم لا تعمل

**الحل:**
```
1. تحقق من IDs الأزرار متطابقة في HTML و JS
2. افتح Console لتشخيص الخطأ
3. تأكد من تحميل الملفات بشكل صحيح
```
