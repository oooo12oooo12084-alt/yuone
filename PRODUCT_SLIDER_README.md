## 📱 منزلق المنتجات - دليل الاستخدام

### ✨ الميزات:

1. **10 منتجات فقط** - يتم عرض أول 10 منتجات من قائمة المنتجات
2. **عرض 4 منتجات** - في كل مرة يتم عرض 4 منتجات فقط
3. **حركة سلسة** - انتقال سلس بين الشرائح مع تأثيرات بصرية رائعة
4. **أزرار التنقل** - 
   - الزر الأيسر: التنقل للشرائح السابقة (يمين في الواجهة العربية)
   - الزر الأيمن: التنقل للشرائح التالية (يسار في الواجهة العربية)
5. **مؤشرات تفاعلية** - للانتقال المباشر إلى أي شريحة
6. **الضغط على اسم المنتج** - عند الضغط على اسم المنتج، يتم تفعيل حدث خاص
7. **الحركة التلقائية** - يتم تشغيل المنزلق تلقائياً كل 6 ثوان

### 📁 الملفات المُضافة:

1. **CCS/product-slider.css** - تنسيقات المنزلق والحركات
2. **JS/product-slider.js** - منطق المنزلق والتفاعلات

### 🔧 التخصيص:

في ملف **JS/product-slider.js**، يمكنك تغيير:

```javascript
new ProductSlider({
    sliderId: 'productSlider',           // معرّف الـ container
    prevBtnId: 'prevBtn',                // معرّف الزر السابق
    nextBtnId: 'nextBtn',                // معرّف الزر التالي
    indicatorsId: 'indicators',          // معرّف المؤشرات
    productsUrl: 'products.json',        // رابط ملف المنتجات
    itemsPerView: 4,                     // عدد المنتجات المعروضة
    autoPlay: true,                      // تشغيل تلقائي
    autoPlayInterval: 6000               // مدة التشغيل التلقائي (ميلي ثانية)
});
```

### 🎨 التصميم:

- العرض: 4 منتجات في الشاشات الكبيرة
- العرض: 3 منتجات في الشاشات المتوسطة (1024px)
- العرض: منتجين في الشاشات الصغيرة (768px)
- العرض: منتج واحد في الهواتف (480px)

### 🖱️ الأحداث:

عند الضغط على المنتج، يتم تفعيل:
- `onProductClick(product)` - عند الضغط على المنتج
- `onProductNameClick(product)` - عند الضغط على اسم المنتج

### 📊 البيانات المطلوبة:

كل منتج في `products.json` يجب أن يحتوي على:
```json
{
    "id": 0,
    "img": "img/product/0.png",
    "name": "اسم المنتج",
    "price": 350,
    "old_price": 450,  // اختياري
    "catetory": "electronics"
}
```
