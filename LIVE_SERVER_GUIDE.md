## 💻 تشغيل الموقع على LIVE SERVER:

### الطريقة 1️⃣: استخدام Live Server (الأسهل)

#### الخطوة 1: تثبيت Live Server
```
1. افتح VS Code
2. اذهب إلى: Extensions (الأيسر)
3. ابحث عن: "Live Server"
4. المطور: Ritwick Dey
5. اضغط: Install
```

#### الخطوة 2: تشغيل Live Server
```
1. افتح مجلد الموقع في VS Code
2. افتح ملف: index.html
3. اضغط كليك يمين على الملف
4. اختر: "Open with Live Server"
```

#### النتيجة:
```
✅ الموقع سيفتح في المتصفح تلقائياً
✅ العنوان: http://127.0.0.1:5500 أو localhost:5500
✅ أي تعديل تجريه سيظهر فوراً (بدون إعادة تحميل)
```

---

### الطريقة 2️⃣: استخدام Python

#### الخطوة 1: افتح Terminal/CMD
```
اضغط: Ctrl + `
أو: اختر Terminal من القائمة العلوية
```

#### الخطوة 2: اكتب الأمر
```
cd f:\dawoanlod\Ecommerce\ Website
python -m http.server 8000
```

#### الخطوة 3: افتح المتصفح
```
اذهب إلى: http://localhost:8000
```

---

### الطريقة 3️⃣: استخدام Node.js

#### الخطوة 1: تثبيت http-server
```
npm install -g http-server
```

#### الخطوة 2: افتح Terminal في المجلد
```
cd f:\dawoanlod\Ecommerce\ Website
http-server -p 8000
```

#### الخطوة 3: افتح المتصفح
```
اذهب إلى: http://localhost:8000
```

## 🔧 التحكم في الحركة:

إذا أردت تغيير سرعة الحركة التلقائية:

افتح: `JS/product-slider.js`

ابحث عن:
```javascript
autoPlayInterval: 10000  // 10 ثوانٍ
```

غيّر الرقم:
- `5000` = 5 ثوانٍ
- `10000` = 10 ثوانٍ
- `15000` = 15 ثانية

---

## 🎨 تغيير لون الأزرار:

افتح: `CCS/product-slider.css`

ابحث عن:
```css
background: #0066cc;  /* الأزرق الحالي */
```

غيّر اللون:
- `#0066cc` = أزرق
- `#FF6B6B` = أحمر
- `#2ECC71` = أخضر
- `#9B59B6` = بنفسجي

## ⚙️ عدد الأسهم:

إذا أردت تحريك أكثر من منتج واحد:

افتح: `JS/product-slider.js`

ابحث عن:
```javascript
itemsToScroll: 1,  // تحريك منتج واحد
```

غيّر الرقم:
- `1` = منتج واحد
- `2` = منتجين
- `3` = 3 منتجات
- `4` = 4 منتجات

## 📱 عرض على الهاتف (من نفس الشبكة):

#### الخطوة 1: احصل على IP الحاسوب
```
في Terminal: ipconfig
ابحث عن: IPv4 Address (مثل: 192.168.x.x)
```

#### الخطوة 2: افتح على الهاتف
```
في متصفح الهاتف اكتب:
http://[IP]:5500

مثال:
http://192.168.1.100:5500
```
