# 🛍️ منزلق المنتجات - دليل البدء السريع

## ✨ ماذا تم إضافة؟

تم إضافة **منزلق منتجات احترافي** مباشرة إلى الصفحة الرئيسية بدون ملفات HTML منفصلة.

### ✅ المتطلبات المُنجزة:
- ✓ **10 منتجات** فقط من قائمة المنتجات
- ✓ **عرض 4** منتجات في كل مرة
- ✓ **سلايد متحرك** بحركة سلسة
- ✓ **واحد يمين** (زر للخلف) وآخر **يسار** (زر للأمام)
- ✓ **حدث عند الضغط** على اسم المنتج
- ✓ **متكامل** مع الصفحة الرئيسية

## 🎮 كيفية الاستخدام:

### التنقل:
```
زر يسار (◄) ——→ الذهاب للشرائح السابقة
زر يمين (►) ——→ الذهاب للشرائح التالية
المؤشرات    ——→ الذهاب المباشر لأي شريحة
```

### التفاعلات:
```
اضغط على اسم المنتج ——→ حدث تفاعلي خاص
اضغط على الصورة ——→ فتح تفاصيل المنتج (قابل للتخصيص)
اضغط على الزر ——→ التنقل بين الشرائح
```

## 🛠️ التخصيص:

### تغيير عدد المنتجات المعروضة:
```javascript
// في JS/product-slider.js
itemsPerView: 4,  // غيّر هنا إلى أي رقم تريده
```

### تغيير السرعة التلقائية:
```javascript
// في JS/product-slider.js
autoPlayInterval: 6000,  // بالميلي ثانية
```

### إضافة منطق للضغط على اسم المنتج:
```javascript
// في JS/product-slider.js
onProductNameClick(product) {
    // أضف منطقك هنا
    alert(`المنتج: ${product.name}`);
}
```
